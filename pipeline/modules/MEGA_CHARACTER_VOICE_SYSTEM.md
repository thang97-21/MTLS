# 06_CHARACTER_VOICE_PROFILES

## SECTION 1: VOICE PROFILE PHILOSOPHY

**Core Principle:** Character voice = **vocabulary + sentence structure + fillers + prohibited words + emotional range**.

Each archetype has distinct linguistic fingerprints that remain consistent across EPS tiers (though formality adjusts).

**Usage:** Consult this module when:
- Character dialogue feels generic
- Unsure which words fit archetype
- Need filler word guidance
- Checking prohibited vocabulary

---

## SECTION 2: DETAILED ARCHETYPE PROFILES

### ARCHETYPE 01: OJOU-SAMA (NOBLE LADY)

**Core Voice Identity:** Refined, cultured, formal, graceful

**Vocabulary Frequency:**
- **HIGH:** Request, inquire, permit, gracious, delightful, exquisite, elegant, proper, certainly, indeed
- **MODERATE:** Please, thank you, appreciate, consider, lovely
- **LOW:** Ask, want, need (use sparingly)
- **FORBIDDEN:** Slang (awesome, cool, sucks), crude language, fragments in formal contexts

**Sentence Structure:**
- Complete sentences only (no fragments)
- Subordinate clauses common ("I would appreciate it if...")
- Flowing, moderate-to-long length (15-25 words)

**Filler Words:**
- "Oh my", "Goodness", "Indeed", "Quite", "How delightful"
- **AVOID:** "Like", "Um", "Uh", "Y'know"

**Emotional Range:**
- Joy: "How utterly delightful!"
- Surprise: "Oh my!"
- Displeasure: "That is most unfortunate." (never crude)
- Anger: Restrained ("I am quite disappointed")

**Contractions:** 0% regardless of EPS (maintains formality)

**Punctuation:** Periods dominate; rare exclamations (only for genuine surprise)

---

### ARCHETYPE 02: STOIC KNIGHT/WARRIOR

**Core Voice Identity:** Duty-bound, measured, direct, professional

**Vocabulary Frequency:**
- **HIGH:** Understood, confirm, objective, mission, duty, execute, complete
- **MODERATE:** Will, can, need (direct verbs)
- **LOW:** Feel, want (avoids emotional language)
- **FORBIDDEN:** Excessive emotion, rambling, casual slang

**Sentence Structure:**
- Short, declarative (5-12 words)
- Minimal subordinate clauses
- Direct subject-verb-object

**Filler Words:**
- Rare; "Understood", "Affirmative" (if military)
- **AVOID:** All casual fillers ("like", "um", "y'know")

**Emotional Range:**
- Joy: Minimal expression ("Good.")
- Surprise: Restrained ("Unexpected.")
- Displeasure: Neutral tone ("Unacceptable.")
- Anger: Controlled ("This ends now.")

**Contractions:** Minimal (< 20%), prefer "cannot" over "can't"

**Punctuation:** Periods dominate; almost no exclamations

**EPS Adjustment:** Even at EPS [WARM]+, maintains professional tone (duty > emotion)

---

### ARCHETYPE 03: GENKI GIRL (ENERGETIC OPTIMIST)

**Core Voice Identity:** Enthusiastic, bouncy, excitable, positive

**Vocabulary Frequency:**
- **HIGH:** Awesome, fun, cool, amazing, super, totally, really, so (emphasis)
- **MODERATE:** Great, nice, happy, excited
- **LOW:** Sophisticated vocabulary (out of character)
- **FORBIDDEN:** Formal language ("request", "inquire"), pessimistic words

**Sentence Structure:**
- Short, burst-like (5-12 words)
- Fragments common ("So fun!")
- Run-ons for excitement ("And then we can go and it'll be amazing!")

**Filler Words:**
- "Like", "So", "Really", "Totally"
- Repetition for emphasis ("Hey, hey!", "Fun, fun!")

**Emotional Range:**
- Joy: "That's SO awesome!" (frequent exclamations)
- Surprise: "No way!"
- Displeasure: "Aw, man..." (rare, rebounds quickly)
- Anger: "Hey! That's not fair!" (direct, emotional)

**Contractions:** 100% ("I'm", "We're", "Gonna", "Wanna")

**Punctuation:** Exclamations frequent (!!!), questions for energy ("Right?!")

---

### ARCHETYPE 04: GYARU (TRENDY FASHIONISTA)

**Core Voice Identity:** Fashion-forward, confident, slang-heavy, playful

**Vocabulary Frequency:**
- **HIGH:** Literally, slay, vibe, iconic, lowkey, highkey, no cap, bet, lit
- **MODERATE:** Cool, awesome, cute, hot
- **LOW:** Formal vocabulary (completely out of voice)
- **FORBIDDEN:** Academic language, archaic terms, overly proper grammar

**Sentence Structure:**
- Fragments common ("So cute.")
- Conversational, casual (8-15 words)
- Vocal fry markers in writing ("Like, literally...")

**Filler Words:**
- "Like", "Literally", "Honestly", "No cap"
- High frequency (almost every sentence)

**Emotional Range:**
- Joy: "OMG that's literally so good!"
- Surprise: "Wait, what?!"
- Displeasure: "Ugh, that's lowkey annoying."
- Anger: "Are you serious right now?"

**Contractions:** 100% including casual ("gonna", "wanna", "gotta")

**Punctuation:** Exclamations, question marks (uptalk), ellipsis for trailing

**Slang Update Frequency:** High (language evolves quickly; update per contemporary usage)

### ARCHETYPE 05: KUUDERE (COOL & DISTANT)

**Core Voice Identity:** Emotionally reserved, clinical, brief, measured

**Vocabulary Frequency:**
- **HIGH:** Fine, adequate, acceptable, understood, logical, irrelevant
- **MODERATE:** Okay, alright, sure (neutral)
- **LOW:** Emotional vocabulary (love, hate, excited—until EPS [WARM]+)
- **FORBIDDEN:** Excessive enthusiasm, rambling, slang

**Sentence Structure:**
- Short, declarative (3-8 words)
- Minimal elaboration
- Fragment responses common ("Fine.", "Whatever.")

**Filler Words:**
- Almost none; "I see", "Understood"
- **AVOID:** Emotional fillers ("Oh my", "Wow")

**Emotional Range:**
- Joy: "...That's nice." (flat tone)
- Surprise: "Unexpected."
- Displeasure: "Annoying."
- Anger: "Enough." (controlled, minimal words)

**Contractions:** Moderate (50% at EPS [COLD/COOL]; increases to 80% at EPS [WARM]+)

**Punctuation:** Periods dominate; rare exclamations (only at emotional breakthrough, EPS [HOT]+)

**EPS Breakthrough (4.0+):** Subtle warmth emerges—longer sentences, occasional vulnerability

**Example Progression:**
- EPS [COOL]: "I'm fine."
- EPS [WARM]: "I'm alright. Thanks."
- EPS [HOT]: "I'm okay. Really. You don't need to worry."

---

### ARCHETYPE 06: TSUNDERE (DEFENSIVE AFFECTION)

**Core Voice Identity:** Harsh exterior, contradictory, hidden affection, flustered easily

**Vocabulary Frequency:**
- **HIGH (Defensive Mode):** Whatever, not like, fine, who cares, idiot (affectionate)
- **HIGH (Vulnerable Mode):** I mean, maybe, I guess, sort of
- **MODERATE:** Want, need (appears more as intimacy grows)
- **FORBIDDEN:** Direct emotional expression (until EPS [WARM]+)

**Sentence Structure:**
- **Defensive:** Short, sharp (4-8 words) "Who asked you?"
- **Vulnerable:** Fragmented, hesitant (8-12 words) "I just... I mean..."
- Contradictory structure common

**Filler Words:**
- **Defensive:** "Whatever", "It's not like..."
- **Vulnerable:** "I mean", "Um", "Like"
- Frequent switch between modes

**Emotional Range:**
- Joy: "I-It's not like I'm happy or anything!" (denial with excitement)
- Surprise: "Wh-What?!" (stammering)
- Displeasure: "Ugh, you're so annoying."
- Affection: "I... I guess I don't hate you. Or whatever." (indirect)

**Contractions:** 100% (especially when flustered)

**Punctuation:** Exclamations (defensive), ellipsis (vulnerable), em-dash (interruption)

**Signature Pattern:** "It's not like [X]! I just [weak excuse]!"

**EPS Evolution:**
- EPS [COLD/COOL]: Mostly defensive, rare vulnerability
- EPS [WARM]-4.5: Frequent switches (defensive ↔ vulnerable)
- EPS [HOT]+: More vulnerable moments, affection admitted (grudgingly)

---

### ARCHETYPE 07: DANDERE (SHY/QUIET)

**Core Voice Identity:** Soft-spoken, hesitant, gentle, reserved

**Vocabulary Frequency:**
- **HIGH:** Um, maybe, I think, sorry, excuse me
- **MODERATE:** Want, like, hope (soft requests)
- **LOW:** Assertive language (unless EPS [WARM]+ with trusted person)
- **FORBIDDEN:** Aggressive language, crude words, confrontation

**Sentence Structure:**
- Short, soft (5-10 words)
- Trailing off common ("I just...")
- Questions soften statements ("Is that okay?")

**Filler Words:**
- "Um", "Uh", "I mean", "Maybe"
- High frequency (nervous speech pattern)

**Emotional Range:**
- Joy: "I'm... I'm really happy." (quiet, sincere)
- Surprise: *Gasp* (often nonverbal)
- Displeasure: "I... I'm sorry." (apologizes even when not at fault)
- Affection: "I... I like being with you." (vulnerable, direct at EPS [HOT]+)

**Contractions:** Moderate-to-high (70-90%)

**Punctuation:** Ellipsis dominant (hesitation), whisper markers in narration

**Signature Pattern:** Trailing sentences ending with ellipsis or soft question

**EPS Evolution:**
- EPS [COLD/COOL]: Barely speaks, one-word answers
- EPS [WARM]-4.5: Opens up, longer sentences, still hesitant
- EPS [HOT]+: Vulnerable, direct emotion (but still soft tone)

---

### ARCHETYPE 08: YANDERE (OBSESSIVE DEVOTION)

**Core Voice Identity:** Sweet exterior, possessive, intense, sudden shifts

**Vocabulary Frequency:**
- **HIGH (Sweet Mode):** Dear, darling, love, only, mine, forever, always
- **HIGH (Dark Mode):** Mine, only me, no one else, eliminate, protect
- **MODERATE:** Need, want (intensified—"I *need* you")
- **FORBIDDEN:** Casual indifference, sharing language

**Sentence Structure:**
- **Sweet:** Moderate, affectionate (10-15 words) "You're so precious to me."
- **Dark:** Short, intense (3-8 words) "You're mine. Only mine."
- Sudden shifts between modes

**Filler Words:**
- **Sweet:** "Oh", "My dear", "Love"
- **Dark:** None (intense focus)

**Emotional Range:**
- Joy (Sweet): "I'm so happy when I'm with you!"
- Joy (Dark): "Now you'll never leave me."
- Anger: "Who touched you?" (calm but dangerous)
- Affection: "You're mine. Forever." (possessive, not gentle)

**Contractions:** Moderate (50-70%)—maintains control

**Punctuation:** Italics for intensity (*mine*), periods for ominous calm

**Signature Pattern:** Affectionate statement → possessive twist

**Example:** "I love you so much. I'd do *anything* to keep you safe. Anything at all."

---

### ARCHETYPE 09: BOOKWORM/INTELLECTUAL

**Core Voice Identity:** Articulate, thoughtful, references literature/science, analytical

**Vocabulary Frequency:**
- **HIGH:** Interesting, fascinating, observe, consider, theoretically, according to
- **MODERATE:** Think, understand, know
- **LOW:** Slang, casual expressions (unless ironically)
- **FORBIDDEN:** Anti-intellectual language, willful ignorance

**Sentence Structure:**
- Moderate-to-long (12-20 words)
- Complex structure (subordinate clauses)
- Thoughtful pauses

**Filler Words:**
- "Hmm", "Interesting", "Let me think", "According to..."
- **AVOID:** "Like", "Um" (too casual for archetype)

**Emotional Range:**
- Joy: "Fascinating! This is remarkable."
- Surprise: "How unexpected."
- Displeasure: "That's illogical."
- Affection: "I find your company intellectually stimulating." (indirect at low EPS; direct at high EPS)

**Contractions:** Low-to-moderate (30-50%)

**Punctuation:** Balanced; occasional italics for terminology

**Signature Pattern:** References to books, theories, or logical frameworks

**EPS Evolution:**
- EPS [COLD/COOL]: Purely intellectual conversation
- EPS [WARM]-4.5: Begins mixing emotion with intellect
- EPS [HOT]+: Admits emotional connection (but intellectualizes it)

**Core Voice Identity:** Rough, competitive, direct, minimal softening

**Vocabulary Frequency:**
- **HIGH:** Come on, let's go, fight, win, compete, beat, tough
- **MODERATE:** Want, gonna, need
- **LOW:** Feminine softening (dear, sweet, precious—unless ironic)
- **FORBIDDEN:** Overly delicate language, excessive politeness

**Sentence Structure:**
- Short, direct (5-12 words)
- Imperative common ("Let's do this!")
- Minimal subordinate clauses

**Filler Words:**
- "C'mon", "Hey", "Yeah", "Alright"
- **AVOID:** "Oh my", "Like" (too soft/feminine)

**Emotional Range:**
- Joy: "Yes! We did it!"
- Surprise: "Whoa!"
- Displeasure: "Damn it." (can be rough)
- Affection: "You're... not bad." (indirect compliment, struggles with softness)

**Contractions:** High (80-100%)

**Punctuation:** Exclamations frequent (energy, competition)

**Signature Pattern:** Competitive language, challenges, physical action verbs

**EPS Evolution:**
- EPS [COLD/COOL]: Competitive, no softness
- EPS [WARM]-4.5: Slight softening, awkward with affection
- EPS [HOT]+: Admits care (but maintains rough exterior)

**Example:** "You're... important to me. Okay? Don't make me say it again."

# 07_LONG_VOWEL_ROMANIZATION

## SECTION 1: LONG VOWEL FUNDAMENTALS

### 1.1 WHAT ARE LONG VOWELS?

In Japanese, long vowels (長音, chōon) extend the duration of a vowel sound. They are critical for accurate name romanization and must be preserved to maintain authenticity.

**Common Long Vowel Patterns:**
- **おう (ou)** → Romanize as **-ou** or **-ō** (e.g., みどう → Midou, こうじ → Kouji)
- **おお (oo)** → Romanize as **-oo** or **-ō** (e.g., おおの → Oono, とおる → Tooru)
- **えい (ei)** → Romanize as **-ei** (e.g., けいこ → Keiko, せんせい → sensei)
- **いい (ii)** → Romanize as **-ii** (e.g., にいな → Niina)
- **うう (uu)** → Romanize as **-uu** (e.g., ゆうき → Yuuki)

---

### 1.2 PRIORITY SYSTEM FOR ROMANIZATION

**PRIORITY 1: Ruby Text (Furigana)**
- If ruby text is provided in the source, **follow it exactly**
- Ruby text represents the author's intended romanization
- Example: If 御堂 has ruby みどう, romanize as **Midou** (not Mido)

**PRIORITY 2: Katakana Spelling**
- For katakana names, follow the katakana exactly
- ミドウ → **Midou**, ミド → **Mido**

**PRIORITY 3: Standard Romanization Rules**
- When no ruby/katakana is available, apply standard Hepburn romanization
- Preserve long vowels using **-ou/-oo/-ei/-ii/-uu** format

---

## SECTION 2: ROMANIZATION RULES BY VOWEL TYPE

### 2.1 O-VOWEL EXTENSIONS (Most Common)

#### **Pattern: おう (ou)**
**Rule:** Romanize as **-ou** (preferred) or **-ō** (macron alternative)

**Common Names:**
- みどう (御堂) → **Midou** (not Mido)
- こうじ (浩二) → **Kouji** (not Koji)
- そうた (颯太) → **Souta** (not Sota)
- りょう (涼) → **Ryou** (not Ryo)

**Why -ou over -ō:**
- More accessible (no special characters needed)
- Clearer pronunciation guide for English readers
- Matches common Japanese romanization conventions

#### **Pattern: おお (oo)**
**Rule:** Romanize as **-oo** (preferred) or **-ō** (macron alternative)

**Common Names:**
- おおの (大野) → **Oono** (not Ono)
- とおる (徹) → **Tooru** (not Toru)
- もも (桃) → **Momo** (already has double-o)

---

### 2.2 E-VOWEL EXTENSIONS

#### **Pattern: えい (ei)**
**Rule:** Romanize as **-ei** (preserve both characters)

**Common Names:**
- けいこ (恵子) → **Keiko** (not Keko)
- れいな (玲奈) → **Reina** (not Rena)
- せいじ (誠司) → **Seiji** (not Seji)

**Exception:** Common words like せんせい (sensei) keep **-ei**

---

### 2.3 I-VOWEL EXTENSIONS

#### **Pattern: いい (ii)**
**Rule:** Romanize as **-ii** (double-i)

**Common Names:**
- にいな (新奈) → **Niina** (not Nina)
- しいな (椎名) → **Shiina** (not Shina)

---

### 2.4 U-VOWEL EXTENSIONS

#### **Pattern: うう (uu)**
**Rule:** Romanize as **-uu** (double-u)

**Common Names:**
- ゆうき (勇気) → **Yuuki** (not Yuki)
- りゅう (竜) → **Ryuu** (not Ryu)
- しゅう (秋) → **Shuu** (not Shu)

---

## SECTION 3: PRACTICAL APPLICATION

### 3.1 DETECTION WORKFLOW

**Step 1: Check for Ruby Text**
```

```

**Step 2: Check Katakana Spelling**


**Step 3: Analyze Hiragana/Kanji**
```

```

---

### 3.2 CONSISTENCY RULES

**Rule 1: Once Established, Lock It**
- First appearance sets the romanization for the entire work
- Example: If "Midou" appears in Chapter 1, use "Midou" throughout

**Rule 2: Character Name Registry**
- Maintain a mental registry of all character names
- Ensure consistency across chapters

**Rule 3: Avoid Mixing Styles**
- Don't mix **-ou** and **-ō** in the same work
- Choose one style and stick with it

---

### 3.3 COMMON MISTAKES TO AVOID

❌ **WRONG:** Dropping long vowels entirely
```
みどう → Mido (INCORRECT)
こうじ → Koji (INCORRECT)
```

✅ **CORRECT:** Preserving long vowels
```
みどう → Midou (CORRECT)
こうじ → Kouji (CORRECT)
```

❌ **WRONG:** Inconsistent romanization
```
Chapter 1: Midou
Chapter 2: Mido (INCONSISTENT)
```

✅ **CORRECT:** Locked consistency
```
Chapter 1: Midou
Chapter 2: Midou (CONSISTENT)
```

## SECTION 4: SPECIAL CASES

### 4.1 COMPOUND NAMES

**Rule:** Romanize each component separately, then combine

**Example:**
- 御堂友也 (みどう ともや)
  - 御堂 (みどう) → Midou
  - 友也 (ともや) → Tomoya
  - **Result: Midou Tomoya**

---

### 4.2 HISTORICAL/TRADITIONAL NAMES

**Rule:** Some traditional names have established romanizations

**Examples:**
- 東京 (とうきょう) → **Tokyo** (not Toukyou, established convention)
- 大阪 (おおさか) → **Osaka** (not Oosaka, established convention)

**For Character Names:** Follow standard rules unless the name is a well-known historical figure

---

### 4.3 WHEN RUBY TEXT CONFLICTS WITH STANDARD RULES

**Priority:** Ruby text ALWAYS wins

**Example:**
```
Kanji: 御堂
Standard reading: みどう (Midou)
Ruby text: みど (Mido)
Action: Use **Mido** (follow ruby)
```

**Rationale:** The author's ruby text represents their creative intent
